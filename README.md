## 仍然适应不了？
---
# 没关系！
> 1. 在开始菜单中键入并运行 ***BEPERFECT*** 命令
> 2. 用右键单击 PerfectWindows 批处理文件，选择 “重命名”
> 3. 在花括号中按照提示键入指定字母：
> ```
> 键入字母 u 以恢复桌面图标、任务栏图标、锁屏等界面元素，并取消浏览器和音量调整的快捷键
> 键入字母 m 以取消鼠标滚轮反向功能
> 键入字母 c 以重新启用 Cortana
> 键入字母 o 以重新启用 OneDrive
> 键入字母 n 以重新启用推送通知与通知中心
> 键入字母 p 以解锁电源选项
> ```
> 4. 按下回车键
> 5. 保存当前所有工作，因为稍后我们将重启电脑
> 6. 用右键单击 PerfectWindows 批处理文件，选择 “以管理员身份运行”
