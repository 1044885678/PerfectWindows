* 现在，你再也不必忍受 Windows 自动更新与自动维护给你带来的不便了，因为 Windows 将不再运行自动更新或自动维护。但是，安全第一，请记得定期手动运行系统更新！
* 为了加快系统的启动与运行，系统将不会自动启动应用。你必须手动启动这些应用。如需更方便地启动它们，请将它们固定至开始菜单。
* 我们调整了你的电脑的电源配置，使得你的电脑在使用电池时耗电更加缓慢，在连接电源时性能更加强劲。

* 你的电脑将只允许你运行位于 C:\Windows 、C:\Program Files 以及 C:\Program Files (x86) 的应用，这可以在不干扰你日常工作与娱乐的前提下极大程度上防止外来的病毒与木马的运行以及流氓软件的静默安装。
* 如果有被你充分信任的应用遭到了阻止，请在开始菜单中键入并运行 ***BEINDANGER*** 命令。注意，此操作会 ***将您的电脑置于不受高强度保护的危险境地！*** 在此状态下，请 ***谨慎对待你的每一个操作*** ，并 ***尽快*** 在开始菜单中键入并运行 ***BESAFE*** 命令以使电脑重新受到高强度的保护！

# 仍然适应不了？
# 没关系！

1. 找到批处理文件
2. 右击批处理文件，选择 “重命名”
3. 在花括号中按照提示键入指定字母：
```
键入字母 u 以恢复桌面图标、任务栏图标、锁屏等界面元素，并取消浏览器和音量调整的快捷键
键入字母 m 以取消鼠标滚轮反向功能
键入字母 c 以重新启用 Cortana
键入字母 o 以重新启用 OneDrive
键入字母 n 以重新启用推送通知与通知中心
键入字母 p 以解锁电源选项
```
4. 按下回车键
5. 保存当前所有工作，因为稍后我们将重启电脑
6. 右击已被你重命名的批处理文件，选择 “以管理员身份运行”
