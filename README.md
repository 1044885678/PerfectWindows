* 这会使电脑进入耗电量极低的睡眠状态。睡眠状态允许你迅速恢复到你先前的工作或娱乐状态。
* 睡眠状态持续 3 个小时后，电脑将自动进入完全不耗电的休眠状态。休眠状态也允许你恢复到你先前的工作或娱乐状态，不过恢复的耗时长于睡眠，但是短于关机之后的开机过程。
* 如果你使用台式电脑并且希望它立即停止耗电，请切断其电源。由于混合睡眠功能已被开启，你先前的工作或娱乐状态并不会因为电脑的断电而丢失。
* 想继续使用电脑？按下电源按钮。为确保安全，无论电脑是从睡眠还是休眠状态恢复，电脑均将要求你输入密码。

# 系统运行卡慢？
# 应用频繁自启？
# 电池消耗太快？
# 系统自动更新太烦？
# 切换任务效率太低？
# 工作娱乐易受干扰？
# 害怕病毒木马侵犯？
# 这烦人的一切都已随风而逝！

* 现在，你再也不必忍受 Windows 自动更新给你带来的不便了，因为 Windows 将不再运行自动更新。但是，安全第一，请记得定期手动运行系统更新！
* 为了缓解系统运行卡慢的问题，我们禁用了 Superfetch、家庭组、系统还原、自动维护、计划任务等有损 Windows 性能的后台服务。
* 为了帮助你更好地掌控应用的启动，除非你手动启动你需要的应用，否则所有应用均不会自动启动。如需更方便地启动你需要的应用，请将它们固定至开始菜单，这也方便你多次启动它。
* 我们调整了你的电脑的电源配置，使得你的电脑在使用电池时耗电更加缓慢，在连接电源时性能更加强劲。 
* 为了防止你在工作与娱乐中遭到不必要的干扰，我们禁用了 Cortana 、OneDrive、推送通知以及通知中心。
* 为了增强电脑的安全系数，我们做了以下工作：
1. 启用、增强并防止你禁用 Windows Defender 防病毒。除非你安装了第三方防病毒软件，否则 Windows Defender 将永远运行。
2. 启用并防止你禁用 Windows Defender 防火墙。
3. 启用并防止你禁用 Windows Defender SmartScreen。
4. 启用并防止你禁用或调整用户账户控制功能。
5. 禁用 U 盘的自动播放功能。
6. **抬高外来可执行文件运行的门槛。如果你在运行可执行文件时遭到了阻拦，请先仔细审核它的来源以及数字签名状态，并仔细考量你是否真的需要运行它。Windows 之所以会遭遇恶意软件和流氓软件的侵害，往往是由于我们在考虑不周全的情况下盲目运行了我们本来不需要的可执行文件。因此，PerfectWindows 抬高了 Windows 运行外来可执行文件的门槛。这个举措为你的冷静思考争取了宝贵的时间，也极大程度上降低了 Windows 受到恶意软件和流氓软件侵犯的可能。如果你发觉某个可执行文件不能完全赢得你的信任，请立即删除它；如果你充分信任并确信自己需要运行某个可执行文件，请将它手动移动至 *X:\TrustedApps* 文件夹（这里的 X: 可以是任意盘符）中，以表示你对它的充分信任。由于 *X:\TrustedApps* 文件夹中的任何可执行文件都不会受到 Windows 阻止，所以请您千万不要将浏览器和下载工具的文件下载路径设置到 *TrustedApps* 目录，也千万不要将压缩软件（*7-Zip、WinRar、WinZip与 2345 好压*）和电子邮件客户端的临时文件夹设置为 *TrustedApps* 文件夹！**

# 仍然适应不了？
# 没关系！

1. 找到批处理文件
2. 右击批处理文件，选择 “重命名”
3. 在花括号中按照提示键入指定字母：
```
键入字母 u 以恢复桌面图标、任务栏图标、锁屏等界面元素，并取消浏览器和音量调整的快捷键
键入字母 m 以取消鼠标滚轮反向功能
键入字母 c 以重新启用 Cortana
键入字母 o 以重新启用 OneDrive
键入字母 n 以重新启用推送通知与通知中心
键入字母 p 以解锁电源选项
```
4. 按下回车键
5. 保存当前所有工作，因为稍后我们将重启电脑
6. 右击已被你重命名的批处理文件，选择 “以管理员身份运行”
