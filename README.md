# 想锁定电脑？
# 想让电脑小憩片刻？
# 想让电脑酣睡一顿？
# 仅需按下电源按钮或合上盖子！
# 剩下的事，无需顾虑！

* 这会使电脑进入耗电量极低的睡眠状态。睡眠状态允许您迅速恢复到您先前的工作或娱乐状态。
* 睡眠状态持续 3 个小时后，电脑将自动进入完全不耗电的休眠状态。休眠状态也允许您恢复到您先前的工作或娱乐状态，不过恢复的耗时长于睡眠，但是短于关机之后的开机过程。
* 如果您使用台式电脑并且希望它立即停止耗电，请切断其电源。由于混合睡眠功能已被开启，您先前的工作或娱乐状态并不会因为电脑的断电而丢失。
* 想继续使用电脑？按下电源按钮。为确保安全，无论电脑是从睡眠还是休眠状态恢复，电脑均将要求您输入密码。

# 系统运行卡慢？
# 电池消耗太快？
# 系统自动更新太烦？
# 切换任务效率太低？
# 工作娱乐易受干扰？
# 害怕病毒木马侵犯？
# 这烦人的一切都已随风而逝！

* 现在，您再也不必忍受 Windows 自动更新给您带来的不便了，因为 Windows 将不再运行自动更新。但是，安全第一，请记得定期手动运行系统更新！
* 为了缓解系统运行卡慢的问题，我们禁用了 Superfetch、家庭组、系统还原、自动维护等有损 Windows 性能的后台服务。
* 我们调整了您的电脑的电源配置，使得您的电脑在使用电池时耗电更加缓慢，在连接电源时性能更加强劲。 
* 为了防止您在工作与娱乐中遭到不必要的干扰，我们禁用了 Cortana 、OneDrive、推送通知以及通知中心。
* [点击此处](https://github.com/szzhiyang/PerfectWindows/blob/master/SecurityREADME.md)以了解 PerfectWindows 是如何增强 Windows 安全性的。

# 仍然适应不了？
# 没关系！

1. 找到批处理文件
2. 右击批处理文件，选择 “重命名”
3. 在花括号中按照提示键入指定字母：
```
键入字母 u 以恢复任务栏图标、任务栏上的工具栏、锁屏等界面元素，并取消浏览器和音量调整的快捷键
键入字母 m 以取消鼠标滚轮反向功能
键入字母 c 以重新启用 Cortana
键入字母 o 以重新启用 OneDrive
键入字母 n 以重新启用推送通知与通知中心
键入字母 p 以解锁电源选项
```
4. 按下回车键
5. 保存当前所有工作，因为稍后我们将重启电脑
6. 右击已被您重命名的批处理文件，选择 “以管理员身份运行”
