# 要是 Windows 也能像 Linux 和 macOS 那样安全无虞就好了，这样我的电脑就不会被防病毒软件拖慢速度，我所信任的软件也不会被无辜误杀了！:smile:
真巧！我们这儿正好有款名叫 [**Checkpoint**](https://github.com/szzhiyang/PerfectWindows/releases/download/1.0/Checkpoint.bat) 的软件可以充分调动 Windows 自带的软件限制策略功能来增强 Windows 的安全性。

:warning: **只要您始终使用最新的 Windows 和软件，并严守以下指导，您的 Windows 即便没有运行防病毒软件，也没有可能中毒！**

**首先根据指引构建防御体系：**

1. 安装 **Checkpoint**。
2. 将浏览器和下载工具的下载目录、解压缩软件的临时解压目录以及即时通讯软件的聊天记录存储目录都设置到 `C:\Checkpoint`、`D:\Checkpoint` 等任意本地磁盘根目录下的 `Checkpoint` 文件夹。**Checkpoint** 只会在系统盘的根目录下自动创建 `Checkpoint` 文件夹；您可以自行在其他磁盘的根目录下创建 `Checkpoint` 文件夹。
3. 重启电脑。

**效果：**

:warning: 您无法直接运行原本不属于您的电脑的可执行文件，也无法双击运行压缩包中的可执行文件。**总之，Windows 只会运行您确信要运行的可执行文件。**

**如果被阻止的文件是您所充分信任的，请遵循以下操作：**

* 如果它是被下载来的，请将其移出 `Checkpoint` 文件夹。
* 如果它位于压缩包中，请将其解压。
* 如果它位于 U 盘，请将其移出 U 盘。
* 如果它位于网络位置，请将其拷贝至本地磁盘。
* 如果它位于移动硬盘或者虚拟光驱，请为移动硬盘或者虚拟光驱分配 `X:`、`Y:` 或者 `Z:` 盘符。

:warning: **您应当只信任具有下列所有特征的可执行文件，否则您的电脑就会陷入危险！**
* 它没有引起您的任何警觉。
* 它拥有有效且正确的数字签名状态。
* 它被大家广泛使用。
* 聊天群里或者论坛上没有对它的负面评价。
* [VirusTotal](https://www.virustotal.com) 和 [腾讯哈勃分析系统](https://habo.qq.com) 都认为它值得您信任。


:warning: **您必须慎重地移动、复制或者解压可执行文件及其所在的文件夹，因为这些操作意味着您已经信任了这些可执行文件！**

:warning: **时刻保持清醒与经常更新软件比什么都重要！**
