* 这会使电脑进入耗电量极低的睡眠状态。睡眠状态允许您迅速恢复到您先前的工作或娱乐状态。
* 睡眠状态持续 3 个小时后，电脑将自动进入完全不耗电的休眠状态。休眠状态也允许您恢复到您先前的工作或娱乐状态，不过恢复的耗时长于睡眠，但是短于关机之后的开机过程。
* 如果您使用台式电脑并且希望它立即停止耗电，请切断其电源。由于混合睡眠功能已被开启，您先前的工作或娱乐状态并不会因为电脑的断电而丢失。
* 想继续使用电脑？按下电源按钮。为确保安全，无论电脑是从睡眠还是休眠状态恢复，电脑均将要求您输入密码。

# 系统运行卡慢？
# 应用频繁自启？
# 电池消耗太快？
# 系统自动更新太烦？
# 切换任务效率太低？
# 工作娱乐易受干扰？
# 害怕病毒木马侵犯？
# 这烦人的一切都已随风而逝！

* 现在，您再也不必忍受 Windows 自动更新给您带来的不便了，因为 Windows 将不再运行自动更新。但是，安全第一，请记得定期手动运行系统更新！
* 为了缓解系统运行卡慢的问题，我们禁用了 Superfetch、家庭组、系统还原、自动维护、计划任务等有损 Windows 性能的后台服务。
* 为了帮助您更好地掌控应用的启动，除非您手动启动您需要的应用，否则所有应用均不会自动启动。如需更方便地启动您需要的应用，请将它们固定至开始菜单，这也方便您多次启动它。
* 我们调整了您的电脑的电源配置，使得您的电脑在使用电池时耗电更加缓慢，在连接电源时性能更加强劲。 
* 为了防止您在工作与娱乐中遭到不必要的干扰，我们禁用了 Cortana 、OneDrive、推送通知以及通知中心。
* 为了增强电脑的安全系数，我们做了以下工作：
1. 启用、增强并防止您禁用 Windows Defender 防病毒。除非您安装了第三方防病毒软件，否则 Windows Defender 将永远运行。
2. 启用并防止您禁用 Windows Defender 防火墙。
3. 启用并防止您禁用 Windows Defender SmartScreen。
4. 增强并防止您禁用或调整用户账户控制功能。
5. 禁用 U 盘的自动播放功能。
6. **提高外来程序和脚本的运行门槛，防止外来程序和脚本的意外运行。如果您在运行程序或脚本时遭到阻止，请仔细确认程序或脚本的来源是否受您信任，并在程序或脚本的属性中检查其数字签名状态。如果您不能充分信任被阻止的程序或脚本，请立即将其删除；如果您充分信任被阻止的程序，请右击该程序后选择 “以管理员身份运行”；如果您充分信任被阻止的脚本，请先为该脚本创建一个快捷方式，再右击该快捷方式，最后选择 “以管理员身份运行”。请您尽量将程序安装在系统盘的 Program Files 文件夹，因为这有助于保护您的电脑安全。另外，请您尽量不要以管理员身份运行尚未被阻止的程序或脚本，因为这可能会将您的电脑置于危险境地！**

# 仍然适应不了？
# 没关系！

1. 找到批处理文件
2. 右击批处理文件，选择 “重命名”
3. 在花括号中按照提示键入指定字母：
```
键入字母 u 以恢复任务栏图标、锁屏等界面元素，并取消浏览器和音量调整的快捷键
键入字母 m 以取消鼠标滚轮反向功能
键入字母 c 以重新启用 Cortana
键入字母 o 以重新启用 OneDrive
键入字母 n 以重新启用推送通知与通知中心
键入字母 p 以解锁电源选项
```
4. 按下回车键
5. 保存当前所有工作，因为稍后我们将重启电脑
6. 右击已被您重命名的批处理文件，选择 “以管理员身份运行”
